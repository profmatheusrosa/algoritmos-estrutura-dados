# Ferramentas e Softwares Recomendados

## Ambientes de Desenvolvimento (IDEs)

### 1. Visual Studio Code (VS Code)
Gratuito, leve e extensível.
- **Extensões Recomendadas:** Python (Microsoft), Pylance, Jupyter, Markdown All in One.

### 2. PyCharm (JetBrains)
IDE robusta específica para Python. Ótima para debugging e testes. A versão Community é gratuita.

### 3. Google Colab / Jupyter Notebook
Ótimo para prototipar algoritmos, plotar gráficos e documentar o raciocínio passo-a-passo. Roda no navegador.

## Visualização de Algoritmos

1.  **Visualgo.net:** O melhor site para visualizar estruturas de dados e algoritmos animados.
2.  **PythonTutor.com:** Permite colar seu código Python e ver a execução linha a linha, visualizando a memória e pilha de chamadas. Essencial para entender Recursão.

## Prática e Desafios

1.  **LeetCode:** Padrão da indústria para preparação de entrevistas.
2.  **HackerRank:** Bom para iniciantes e testes de empresas.
3.  **CodeWars:** Exercícios divertidos baseados na comunidade (Katas).

## Diagramação

1.  **Excalidraw:** Ferramenta estilo quadro branco, ótima para desenhar grafos e árvores manualmente.
2.  **Mermaid.js:** Criar diagramas usando código (Markdown). Útil para documentação no GitHub.

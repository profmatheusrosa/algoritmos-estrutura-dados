# Módulo 09: Exercícios - Design de Sistemas

## Exercício Prático 1: Encurtador de URL (TinyURL)

Projete (em texto/pseudocódigo) um sistema como o bit.ly.
1.  **Requisitos:** Receber uma URL longa, gerar uma curta, redirecionar quando acessada.
2.  **Estimativa:** 1 milhão de novos links por dia.
3.  **Perguntas:**
    - Qual banco de dados usar? (SQL vs NoSQL)
    - Como gerar a chave curta (Hash vs Contador)?
    - O que acontece se dois usuários encurtarem a mesma URL?

## Exercício Prático 2: Feed de Notícias (Instagram)

Como você armazenaria o feed de um usuário?
Opção A: Gerar o feed na hora que o usuário abre o app (Pull Model).
Opção B: Pré-calcular o feed e salvar em cache quando os amigos postam (Push Model).

Discuta os prós e contras de cada abordagem.

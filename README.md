# Curso: Algoritmos e Estrutura de Dados

Um curso completo cobrindo os fundamentos essenciais da Ciência da Computação, desde a análise de complexidade até algoritmos avançados de grafos e design de sistemas. Focado em implementação prática com **Python**.

## Visão Geral do Curso

- **Carga Horária Total:** 40 horas
- **Duração por Aula:** ~4.5 horas (média)

## Pré-requisitos e Ambiente de Desenvolvimento

- **Python 3.8+** instalado.
- Um editor de código (VS Code, PyCharm ou Jupyter Notebook).
- Conhecimentos básicos de lógica de programação (variáveis, loops, condicionais).

## Integridade Acadêmica

Incentivamos o esforço próprio. A cópia de código sem entendimento prejudica apenas o seu próprio aprendizado. O uso de IA (como ChatGPT/Claude) é permitido e encorajado como **ferramenta de apoio** (tutor), mas não para gerar a solução final sem que você a compreenda. Os projetos foram desenhados para serem desafiadores e construir seu portfólio.
